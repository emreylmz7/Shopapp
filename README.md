# ShopApp

**ShopApp** is a comprehensive e-commerce application built with .NET Core, featuring a user interface for browsing products, cart functionality, and purchasing items. The admin panel provides capabilities for product management. This application utilizes MVC and Entity Framework Core to offer flexibility and robust features.

## Features

- **User Interface**: A modern and intuitive UI for browsing products, viewing product details, and managing the shopping cart.
- **Cart Functionality**: Add products to the cart, view cart details, and proceed to checkout.
- **Purchasing**: Complete transactions and manage orders through a streamlined purchasing process.
- **Admin Panel**: Manage products, categories, and orders with a dedicated admin interface.
- **MVC Architecture**: Built using the Model-View-Controller (MVC) pattern for organized and scalable code.
- **Entity Framework Core**: Utilizes EF Core for flexible and efficient data access and management.

## Installation

To set up and run the project locally, follow these steps:

### Prerequisites

- **.NET Core SDK**: Version 6.0 or higher.
- **MySQL Server**: Required for the database. You can use SQL Server Express for local development.
